# 👻 Flappy Ghost

A simple **Flappy Bird-style game built with Pygame**.  
Control the ghost, avoid obstacles, and survive as long as possible to get the highest score.

---

## ✨ Highlights
- 🐤 Smooth movement with gravity system
- 🧱 Random pipe generation
- 💥 Collision detection (pipes & ground)
- 🔊 Sound effects (flap & hit)
- ⏱️ Countdown before game starts
- 📊 Score & High Score tracking
- 🔁 Instant restart without restarting program

## 🧠 How It Works
1. The game starts with a short countdown  
2. Press `SPACE` to make the bird (ghost) fly  
3. Pipes are generated at random heights  
4. Avoid hitting pipes or the ground  
5. Score increases over time  
6. Game ends when collision occurs  

---

## 🎮 Controls

| Action | Key |
|--------|-----|
| Fly | `SPACE` |
| Restart | `SPACE` |
| Exit | Close window |

---

## 🛠 Requirements
- Python 3.x
- Pygame

## 🚀 How to Run

1. Clone repository:
```bash
git clone https://github.com/putrilia12/flappy-bird.git
cd flappy-bird
```

2. Run the game:
```bash
python main.py
```
